# Particle Universe

An interactive, high-performance particle system website built with HTML5 Canvas and Vanilla JavaScript.

## Features
- **Dynamic Background**: Thousands of particles connecting and reacting to each other.
- **Interactive Clock**: A live-updating date and time display formed entirely by particles in the top-right corner.
- **Mouse Interaction**: Particles (including the clock) react to mouse movement, scattering when hovered over.
- **Premium Aesthetics**: Dark-themed design with smooth gradients and modern typography.

## How to Run
1. **Directly**: Open `index.html` in any modern web browser.
2. **Via Python Server**:
   ```bash
   uv run main.py
   ```
   This will start a local server at `http://localhost:8000` and automatically open your browser.

## Built With
- HTML5 / CSS3
- Vanilla JavaScript (Canvas API)
- Python (for the optional dev server)
